Application Server

Recently Apple has upgraded its server for Mac OS X Server 10.3. In this new update includes WebObjects 5.2.3, which also has support from Java 1.4.2. Other components of the Java Application server are updated such as the JBoss, which is upgraded to version 3.2.3 and contains new security features and fixes. The Tomcat is updated to version 4.1.29 and is integrated with the JBoss distribution.

Job/Utility Server

The most recent utility server updated for Apple is the Airport Utility 1.3.4 for IOS. IOS 7.1 is required to use this server or any other updated version of IOS. There are a lot of products that are needed for this software to work such as the airport Express 802.11n(2nd Generation), Airport Extreme 802.11(1st Generation), Airport Express 802.11n(1st Generation), etc.

Open Source Software

Apple is the first major computer company to make Open Source development a major part of its software strategy and is continuing to use this same strategy. The major components of Mac OS X are all under Apple’s Open Source license. One of the Open source projects included is the Apache, which can only be used in version 2.2.19. Another open source project is glibtool which is used under version 2.2.10 and can also be used in Xcode Tools.

Deployment Strategies

With the Lion Server you can create Netinstall or NetRestore images for mass deployment of OS X Lion Server and Lion Server. Netinstall images can be used to download OS X and Lion Server and other various other packages. You can use Netinstall images to upgrade the volume of a previous version of Mac OS X without deleting it’s first version. NetRestore is used to delete the destination volume as well as making a block copy of a pre-installed system over the network. The NetRestores are much faster than NetImages and is preferable.

Security

For Apple one of the new software that is out for security of your product is the OS X Yosemite v10.10.2 and Security Update 2015-001. It has several components that come with this such as an AFP Server, which is a remote attacker, which can determine all the network addresses in the system. Another component is the bash, which allows local attackers to execute arbitrary code. It also comes with a Bluetooth, which is a an application that lets you execute arbitrary code with system privileges.
